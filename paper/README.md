# Paper

LaTeX source for "Cheap Attacks, Cheap Defenses: An Empirical Audit of
Safeguard Robustness in Open-Weight LLMs."

## Build

```bash
cd paper
make           # compiles main.pdf
make view      # open main.pdf
make clean     # remove build artifacts
```

Requires `pdflatex` and `bibtex` (TeXLive, MacTeX, or MiKTeX).

## Workflow

1. After the cloud sweep completes, run:
    ```bash
    advsafe-report --results results/sweep --output paper/results/
    ```
2. The report CLI writes:
    - `paper/results/RESULTS.md` — markdown summary to copy into `main.tex`
    - `paper/results/fig_pareto.png` — headline figure (referenced by main.tex)
    - `paper/results/h1.json`, `h2.json`, `h3.json` — hypothesis verdicts
    - `paper/results/safety_half_lives.json` — SHL per (model, defense)
    - `paper/results/defense_composability.json` — DC per (model, attack)
3. Replace the `\TODO` placeholders in `main.tex` with values from those files.
4. Run `make` to compile.

## Structure

- `main.tex` — paper body, with section structure pre-built
- `references.bib` — bibliography (BibTeX)
- `results/` — generated by `advsafe-report` (gitignored)
- `figures/` — manually-curated figures (currently empty)
- `tables/` — manually-curated tables (currently empty)
- `Makefile` — build targets
